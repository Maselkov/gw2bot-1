The default prefix is $

Use commands with multiple options with a space in between: $key add

  • account

    Information about your account

  • bosses

    Lists all the bosses you haven't killed this week

  • cats

    Displays cats you haven't collected

  • changelog

    List of recent changes to the bot

  • character

    Character related commands

    birthdays

    Lists days until each of your character's birthdays

    build

    Displays the build of given character

    gear

    Displays the gear of given character

    info

    Info about the given character

    list

    Lists all your characters

    pvpbuild

    Displays the build of given character

    togglepublic

    Toggle your character's status to public

    wvwbuild

    Displays the build of given character

  • daily

    Commands showing daily things

    all

    Show today's all dailies

    fractal

    Show today's fractal dailies

    psna

    Show today's Pact Supply Network Agent locations

    pve

    Show today's PvE dailies

    pvp

    Show today's PvP dailies

    wvw

    Show today's WvW dailies

  • dulfy

    Search dulfy.net

  • et

    The event timer. Shows upcoming world bosses

  • gem

    Commands related to gems

    price

    Lists current gold/gem exchange prices

    track

    Receive a notification when cost of 400 gems drops below the specified cost

  • hotet

    The HoT event timer. Shows phases of HoT maps

  • guild

    Guild related commands

    info

    Information about general guild stats

    members

    Get list of all members and their ranks

    treasury

    Get list of current and needed items for upgrades

  • gw2wiki

    Search the Guild Wars 2 wiki

  • key

    Commands related to API keys

    add

    Adds your key and associates it with your discord account

    info

    Information about your API key

    remove

    Removes your key from the list

  • prefix

    Sets bot's prefixes for this server

  • pvp

    Commands related to PvP

    professions

    Information about your pvp profession stats

    stats

    Information about your general PvP stats

  • search

    Search your account for items

  • skillinfo

    Information about a given skill

  • tp

    Commands related to Trading Post

    current

    Show current selling/buying transactions

    price

    Checks price of an item

  • wallet

    Commands related to wallet

    currencies

    Returns a list of all currencies

    currency

    Info about a currency. See [p]wallet currencies for list

    maps

    Shows map-specific currencies

    show

    Shows most important currencies in your wallet

    tokens

    Shows instance-specific currencies

  • wvw

    Commands related to WvW

    info

    Info about a world. If none is provided, defaults to account's world

    worlds

    List all worlds

  • li

    Shows how many Legendary Insights you have

  • help

    Shows the commands menu

Notifiers

  • updatenotifier

    For setting up notifcations whenever the game updates

    channel

    Sets the channel to send the update announcement

    toggle

    Toggles sending game update notifications

  • newsfeed

    For setting up automatic guildwars2.com news feed

    channel

    Sets the channel to send the news to

    toggle

    Toggles posting news

  • dailynotifier

    Sends a list of dailies to specified channel

    channel

    Sets the channel to send the dailies to

    toggle

    Toggles posting dailies at server reset

    autodelete

    Toggles autodeleting last day's daily notification

    autopin

    Toggles autopinning daily notifications

    categories

    Choose which categories to display on daily notifications
